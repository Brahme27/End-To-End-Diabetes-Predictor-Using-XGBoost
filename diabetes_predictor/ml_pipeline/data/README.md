# ML Pipeline - Data Directory

This directory contains the datasets for the diabetes risk prediction model.

## Structure

- `raw/` - Original, unprocessed data files
- `processed/` - Cleaned and preprocessed data ready for training

## Data Files

The data will be generated automatically by running the training scripts.

### Raw Data
- `diabetes_raw.csv` - Original dataset with all features

### Processed Data
- `diabetes_processed.csv` - Preprocessed dataset ready for model training
